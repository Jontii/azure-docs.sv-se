# <a name="overview-of-the-pci-dss-v321-blueprint-sample"></a>Översikt över PCI-DSS v3.2.1-skissexemplet

PCI-DSS v3.2.1-skissexemplet är en uppsättning principer som hjälper till dig att uppnå PCI-DSS v3.2.1-efterlevnad. Den här skissen hjälper kunderna att styra molnbaserade miljöer med PCI-DSS-arbetsbelastningar.
PCI-DSS-skissen distribuerar en grundläggande uppsättning principer för Azure-distribuerade arkitekturer som kräver den här ackrediteringen.

## <a name="control-mapping"></a>Kontrollmappning

Avsnittet om kontrollmappning innehåller information om de principer som ingår i det här initiativet och hur dessa principer hjälper till att uppfylla olika kontroller som definieras i PCI-DSS v3.2.1. Resurser som tilldelats till en arkitektur utvärderas för icke-kompabilitet med tilldelade principer av Azure Policy.

Kontrollera dina Azure-miljöers efterlevnad i instrumentpanelen för enhetsefterlevnad i Azure Policy när du har tilldelat den här skissen.
