# <a name="overview-of-the-fedramp-high-blueprint-sample"></a>Översikt över skissexempel för FedRAMP High

Skissexemplet för FedRAMP High ger styrningsskydd via [Azure Policy](../../../policy/overview.md) som hjälper dig att utvärdera specifika FedRAMP High-kontroller. Den här skissen hjälper kunderna att distribuera en grundläggande uppsättning principer för all Azure-distribuerad arkitektur som måste implementera FedRAMP High-kontroller.

## <a name="control-mapping"></a>Kontrollmappning

Avsnittet om kontrollmappning innehåller information om de principer som ingår i den här skissen och hur dessa principer täcker olika kontroller i FedRAMP High. Resurser som tilldelats till en arkitektur utvärderas för icke-kompabilitet med tilldelade principer av Azure Policy. Mer information finns i [Azure Policy](../../../policy/overview.md).
