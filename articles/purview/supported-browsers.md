# <a name="supported-browsers"></a>Webbläsare som stöds 

Azure avdelningens kontroll stöder följande webbläsare. Vi rekommenderar att du använder den senaste webbläsaren som är kompatibel med ditt operativsystem. 

* Microsoft Edge (senaste versionen)
* Safari (senaste versionen, endast Mac)
* Chrome (senaste versionen)
* Firefox (senaste versionen)

## <a name="chrome-incognito-mode"></a>Chrome Incognito-läge

 Chrome Incognito blockerar cookies från tredje part måste inaktive ras för att avdelningens kontroll Studio ska fungera.

:::image type="content" source="./media/supported-browsers/incognito-chrome.png" alt-text="Skärm bild som visar Chrome.":::

## <a name="chromium-edge-inprivate-mode"></a>InPrivate-läge för krom Edge

Krom Edge InPrivate med strikt spårnings skydd måste inaktive ras för att avdelningens kontroll Studio ska fungera.

:::image type="content" source="./media/supported-browsers/incognito-edge.png" alt-text="Skärm bild som visar Edge.":::
