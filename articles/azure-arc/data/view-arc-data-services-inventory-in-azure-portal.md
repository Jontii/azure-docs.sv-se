# <a name="view-inventory-of-your-instances-in-the-azure-portal"></a>Visa inventering av dina instanser i Azure Portal

När du har laddat upp dina [mått, loggar](upload-metrics-and-logs-to-azure-monitor.md)eller [användning](view-billing-data-in-azure.md)kan du Visa din instans som en Azure-resurs från Azure Portal. För att göra det navigerar du till den särskilda URL: en [https://portal.azure.com](https://portal.azure.com) och följer skärm bilderna nedan

1. Gå till alla tjänster
1. Sök efter din typ av databas instans typer
1. Lägg till den i dina favoriter
1. Klicka på instans typ från det vänstra navigerings fältet
1. Visa dina instanser i samma vy som dina andra Azure SQL-eller Azure PostgreSQL-storskaliga resurser (Använd filter för att få en detaljerad vy)
