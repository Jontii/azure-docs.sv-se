1. Kopiera kommandot till en text redigerare.
2. Gör följande ändringar i kommandot där det behövs:
    1. Ersätt värdet `<your-text-analytics-key-here>` med din nyckel.
    2. Ersätt den första delen av fråge-URL: en `<your-text-analytics-endpoint-here>` med din egen slut punkts-URL.
3. Öppna ett kommandotolksfönster.
4. Klistra in kommandot från textredigeraren i kommandotolkens fönster och kör sedan kommandot.
