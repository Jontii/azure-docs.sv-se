# <a name="quickstart-use-the-rest-apis-to-customize-your-solution"></a>Snabb start: Använd REST-API: er för att anpassa din lösning

Kom igång med REST API Metrics Advisor. Följ de här stegen för att installera paketet och prova exempel koden för grundläggande uppgifter.

Använd mått rådgivare för att utföra:

* Lägga till en datafeed från en data Källa
* Kontrol lera inmatnings status
* Konfigurera identifiering och aviseringar 
* Fråga efter avvikelse identifierings resultat
* Diagnostisera avvikelser
