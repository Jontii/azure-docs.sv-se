# <a name="monitor-and-manage-wunderlist-in-azure-logic-apps"></a>Övervaka och hantera Wunderlist i Azure Logic Apps

Från och med den 6 maj 2020 [stänger Wunderlist-tjänsten ned](https://www.wunderlist.com/) och efterfrågade kunder för att [migrera sina data och börja använda Microsoft för att göra det i stället](https://www.wunderlist.com/blog/goodbye-from-wunderlist/). Du kan använda [Microsoft för att göra-anslutningen](/connectors/todo/) för att ersätta den föråldrade [Wunderlist-anslutningen](https://preview.flow.microsoft.com/connectors/shared_wunderlist/wunderlist/).
