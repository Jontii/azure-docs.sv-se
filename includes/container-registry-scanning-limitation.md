> [!NOTE]
> Azure Security Center kan för närvarande inte utföra [skanning av bild sårbarhet](../articles/security-center/azure-container-registry-integration.md?toc=/azure/container-registry/toc.json&bc=/azure/container-registry/breadcrumb/toc.json) i ett register som begränsar åtkomsten till privata slut punkter, valda undernät eller IP-adresser. Instanser av vissa Azure-tjänster, inklusive Azure DevOps Services och Azure Container Instances, kan inte heller komma åt en nätverks begränsad behållar register.
