> [!TIP]
> Du kan använda kommandot [Docker images](https://docs.docker.com/engine/reference/commandline/images/) för att visa en lista över hämtade behållar avbildningar. Följande kommando visar till exempel ID, lagrings plats och tagg för varje Hämtad behållar avbildning, formaterad som en tabell:
>
>  ```
>  docker images --format "table {{.ID}}\t{{.Repository}}\t{{.Tag}}"
>
>  IMAGE ID         REPOSITORY                TAG
>  <image-id>       <repository-path/name>    <tag-name>
>  ```
