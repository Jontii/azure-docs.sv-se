## <a name="more-information-about-bots"></a>Mer information om robotar

Om du vill ha mer information om hur du använder den här tjänsten med robotar börjar du med följande resurser:

|Resurs|Syfte|
|--|--|
|[Azure bot-tjänst](https://dev.botframework.com/)|Azure bot service tillhandahåller en komplett molnbaserad webb tjänst med en bot-slutpunkt. Tjänsterna använder [bot Framework](https://github.com/Microsoft/botframework), som är tillgängligt på flera språk.|
|[Bot Framework](https://github.com/Microsoft/botframework)|Microsoft bot Framework är en omfattande plattform för att skapa konversations AI-upplevelser i företags klass.|
|[Bot Framework-emulator](https://github.com/Microsoft/botframework#Bot-Framework-Emulator)|Bot Framework-emulatorn är ett plattforms oberoende Desktop-program som gör det möjligt för bot-utvecklare att testa och felsöka robotar som skapats med bot Framework SDK. Du kan använda bot Framework-emulatorn för att testa robotar som körs lokalt på datorn eller för att ansluta till robotar som körs via fjärr anslutning.|
|[Bot verktyg](https://github.com/Microsoft/botbuilder-tools)|Verktygen för bot Framework är en samling plattforms oberoende kommando rads verktyg som utformats för att kunna använda arbets flödet från slut punkt till slut punkt för bot-utveckling. |
|[Robot Builder-exempel](https://github.com/Microsoft/BotBuilder-Samples)|Fullständigt utvecklade robot exempel är utformade för att illustrera scenarier som du behöver implementera för att bygga fantastiska robotar.|
