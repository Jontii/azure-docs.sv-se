Det enklaste sättet att skapa ett Azure Storage-konto är genom att använda [Azure Portal](https://portal.azure.com). Läs mer i [Skapa ett lagringskonto](../articles/storage/common/storage-account-create.md).

Du kan även skapa ett Azure-lagringskonto med hjälp av [Azure PowerShell](/powershell/module/az.storage/) eller [Azure CLI](../articles/storage/blobs/storage-quickstart-blobs-cli.md).

Om du inte vill skapa ett lagrings konto just nu kan du också använda Azure Storage emulatorn för att köra och testa din kod i en lokal miljö. Mer information finns i [använda Azure Storage emulatorn för utveckling och testning](../articles/storage/common/storage-use-emulator.md).
